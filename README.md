# LinuxMediaServerPHP

## You can do the following:
 - Download youtube videos
 - Play, pause etc. 
 - Create categories, subpages
 
 ## Created with:
  - PHP
  - jQuery
  - MySql
  - CSS Bootstrap
  - HTML5

##
 * mplayer is used for playing the videos.
 * youtube-dl is used for downloading the videos.
